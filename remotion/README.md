# DIFARYX Google AI Challenge Video

This folder is an isolated Remotion video-generation layer. It does not change the Vite app routes or production app behavior.

## Preview

```powershell
npm run video:studio
```

Open the `GoogleAIChallengeVideo` composition in Remotion Studio.

## Render

```powershell
npm run video:render
```

The MP4 is written to:

```text
remotion/out/difaryx-google-ai-challenge.mp4
```

## Composition Settings

- Duration: 60 seconds
- FPS: 30
- Frames: 1800
- Resolution: 1920x1080
- Aspect ratio: 16:9
- Audio: none
